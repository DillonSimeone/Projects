Splatter Setting

Fancy Tool Tip - Splatter Setting

Setting for the silly canvas onclick events for all of the account pictures.

  • Color: The colors for the splatters.
  • Shades: Possible shades of colors. (255 > && < 0)
  • Amount: How much splatter particles to generate on click. ( > 1 )
  • Angle: Spacing between particles. ( > 1 )
  • AngleXdelta: Affects the distance of particles on the x-axis. ( > 1 )
  • AngleYdelta: Affects the distance of particles on the y-axis. ( > 1 )
  • Randomness: Affects the distance particles travels on either axises. ( > 1 )
  • Size: Affects the size of particles. ( > 1 )
  • SizeDelta: Affects how big the particles will be ( > 1)
  • Life: Affects the size and render time for particles. ( > 1)
